componentDidMount()

When the component mounts, we fetch the article we have to display. The url that triggers displaying this Screen is like this: /article/:articlenid For example it can be: /article/27 The number is the node id of the article in the Drupal backend. This component uses the service ArticleService for it to fetch that specific article.
