Class: HomeScreen

HomeScreen()

Home Screen. Header and Footer are not here.

Here you find the presentation text ("Hello, my name is Emanuele Santanch√©. I am a Web Developer and build websites and web applications using JavaScript, React, Angular, WordPress, Drupal, HTML5, CSS3, CSSinJS.") and the lists of articles separated by topic.

The TopicService provides details about each topic.

These details are used to show the avatar for each list of articles, the title, the subtitle and the link to more articles for the same topic.

On the home page only sticky articles are shown. Articles are flagged as sticky in the backend.
