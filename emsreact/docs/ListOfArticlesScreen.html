Class: ListOfArticlesScreen

ListOfArticlesScreen(topic, sticky)

This Screen shows a list of articles. It's supposed to use the service ArticleService to retrieve them from the backend.

If the prop sticky is true, only the articles that are flagged as sticky on the backend will be retrieved.

Constructor

new ListOfArticlesScreen(topic, sticky)

Parameters:
Name Type Description
topic string

The topic of articles to show. Like "How I work", "The tools I use", "Success stories", etc.
sticky boolean

True if only sticky articles should be displayed
